# بوت عبد الزهرة جونير 🤖

بوت تيليغرام عربي للمجموعات بشخصية "جونير" العراقية الساخرة، مع نظام اقتصاد، ألعاب، حماية، ولوحة تحكم مطور كاملة داخل تيليغرام.

## ما تم بناؤه في هذه النسخة الأولى

- ✅ شخصية جونير + الذكاء الاصطناعي (Claude API) مع سياق المحادثة
- ✅ نظام الردود الذكية المخصصة (نص/صور/فيديو/ملصقات/ملفات)
- ✅ نظام النقاط والخبرة + الشهرة + توب 10
- ✅ نظام الاقتصاد الأساسي (محفظة، بنك، راتب يومي، تحويل)
- ✅ لوحة تحكم المطور بأزرار Inline
- ✅ نظام حماية من الروابط + تحذيرات + كتم تلقائي
- ✅ نظام ترحيب تلقائي بالأعضاء الجدد
- ✅ مثالين على الألعاب (تخمين الرقم، حجر ورقة مقص) قابلين للتوسعة

> هذا أساس قوي وقابل للتشغيل فعلياً. الأنظمة الكبيرة المتبقية من الخطة الأصلية (الزواج الوهمي، العقارات، السيارات، كل الألعاب العشرة، نظام الرتب الكامل، الاقتراحات/الشكاوى...) جاهزة للإضافة بنفس النمط — أخبرني بالتالي الذي تريد إضافته.

## 1) خطوات الحصول على المفاتيح المطلوبة

### أ) توكن البوت (BOT_TOKEN)
1. افتح تيليغرام وتكلم مع [@BotFather](https://t.me/BotFather)
2. اكتب `/newbot` واتبع التعليمات (اسم البوت ثم username ينتهي بـ bot)
3. سيعطيك توكناً شكله: `123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11`

### ب) الـ ID الخاص بك (DEVELOPER_ID)
1. تكلم مع [@userinfobot](https://t.me/userinfobot)
2. سيعطيك رقم ID الخاص بحسابك

### ج) مفتاح Anthropic (ANTHROPIC_API_KEY)
1. سجل بحساب على [console.anthropic.com](https://console.anthropic.com)
2. من قسم API Keys أنشئ مفتاحاً جديداً وانسخه
3. تحتاج إضافة رصيد بسيط (Billing) لتفعيل الاستخدام

### د) أهم خطوة لتيليغرام: صلاحيات البوت بالمجموعة
بعد إضافة البوت لأي مجموعة، يجب ترقيته لـ **مشرف (Admin)** فيها، وتفعيل صلاحية:
- حذف الرسائل (لنظام الحماية)
- حظر/كتم الأعضاء (لنظام التحذيرات)

## 2) الرفع على Railway (خطوة بخطوة)

### الخطوة 1: رفع الكود على GitHub
1. أنشئ مستودع (repository) جديد على GitHub
2. ارفع له كل ملفات المشروع (مجلد `abdulzahra_bot` بالكامل)

```bash
cd abdulzahra_bot
git init
git add .
git commit -m "أول نسخة من بوت جونير"
git branch -M main
git remote add origin https://github.com/USERNAME/REPO_NAME.git
git push -u origin main
```

### الخطوة 2: إنشاء مشروع على Railway
1. افتح [railway.app](https://railway.app) وسجل دخول (يفضل بحساب GitHub مباشرة)
2. اضغط **New Project** → **Deploy from GitHub repo**
3. اختر المستودع الذي رفعت له الكود
4. Railway سيكتشف ملف `Procfile` تلقائياً ويبدأ embed التشغيل كـ worker

### الخطوة 3: إضافة قاعدة بيانات PostgreSQL
1. بنفس المشروع على Railway، اضغط **New** → **Database** → **Add PostgreSQL**
2. سينشئ Railway قاعدة بيانات تلقائياً ويعطيها متغير اسمه `DATABASE_URL`

### الخطوة 4: ضبط متغيرات البيئة (Environment Variables)
بإعدادات الخدمة (Service) التي تشغل البوت، اذهب لتبويب **Variables** وأضف:

| المتغير | القيمة |
|---|---|
| `BOT_TOKEN` | التوكن من BotFather |
| `ANTHROPIC_API_KEY` | مفتاح Anthropic |
| `DEVELOPER_ID` | رقم الـ ID الخاص بك |
| `DATABASE_URL` | اربطه بمتغير قاعدة البيانات (Railway يوفر زر "Reference" لربطه تلقائياً من خدمة Postgres) |

> ملاحظة: لا تحتاج لإضافة `DATABASE_URL` يدوياً — اضغط على المتغير واختر "Add Reference" لتربطه تلقائياً بخدمة Postgres التي أنشأتها.

### الخطوة 5: التشغيل
بعد حفظ المتغيرات، Railway سيعيد تشغيل الخدمة تلقائياً. تابع **Logs** للتأكد من ظهور:
```
✅ تم الاتصال بقاعدة البيانات بنجاح
✅ تم التحقق من جميع الجداول/إنشاؤها بنجاح
🚀 البوت جاهز للعمل
```

إذا ظهرت هذي الرسائل، يعني البوت شغال! جربه بإضافته لمجموعة واكتب `جونير شلونك`.

## 3) التشغيل المحلي (للتجربة قبل الرفع)

```bash
cd abdulzahra_bot
python -m venv venv
source venv/bin/activate   # على ويندوز: venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env
# عدّل ملف .env وضع فيه القيم الحقيقية
python main.py
```

## 4) الأوامر المتوفرة حالياً

**عامة:** `/start`، `جونير [سؤال]`، `ايدي`، `مالك`، `مطور`، `توب`، `الشهرة`

**الاقتصاد:** `فلوسي`/`رصيدي`/`محفظتي`، `راتب`، `ايداع [مبلغ]`، `سحب [مبلغ]`، `رصيد البنك`، `تحويل [مبلغ]` (بالرد)

**الألعاب:** `تخمين الرقم`، `حجر ورقة مقص [حجر/ورقة/مقص]`

**المطور فقط:** `/dev` (اللوحة), `حظر مجموعة [ID]`, `فك حظر مجموعة [ID]`, `تفعيل مجموعة [ID]`, `تعطيل مجموعة [ID]`, `تشغيل الذكاء الاصطناعي`, `ايقاف الذكاء الاصطناعي`, `إذاعة [نص]`
